# Mocklab Frontend

React frontend application for managing mock API responses.

## Features

- Create, read, update, and delete mock responses
- Toggle mock responses active/inactive
- Filter and search mock responses
- Modern UI with PrimeReact components
- Responsive design

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:3000`

### Build

```bash
npm run build
```

## Tech Stack

- React 19
- PrimeReact 10
- React Router DOM 7
- Vite 7
